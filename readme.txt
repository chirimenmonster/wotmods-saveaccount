# SaveAccount- yet another save password
========
## Description
Mod SaveAccount is yet another save password, and written in python.
don't need anymore gui_settings.xml, so robust againt change original gui_settings.xml

## Install
To install, copy .wotmod file (saveaccount-1.2.wotmod) to your WoT mods folder
(ex. C:\Games\World_of_Tanks\mods\0.9.17.1).

## Support and Resources
* [ASIA Forum](http://forum.worldoftanks.asia/index.php?/topic/69879-/)
* [GitHub](https://github.com/chirimenmonster/wotmods-saveaccount)

========
## 説明
SaveAccount はパスワード保存を行う mod で、Python で記述されています。
gui_settings.xml は必要としないので、元の gui_settings.xml が更新されても問題ありません。

## インストール
.wotmod ファイル (saveaccount-1.1.wotmod) を WoT クライアントの mod フォルダ
(ex. C:\Games\World_of_Tanks\mods\0.9.17.1)　にコピーして下さい。
